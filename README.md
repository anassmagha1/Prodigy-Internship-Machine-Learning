Prodigy Infotech – Machine Learning Internship Tasks

Task 1:
Build a Linear Regression model to predict house prices based on features such as square footage, number of bedrooms, and number of bathrooms.

Task 2:
Implement a K-Means clustering algorithm to segment retail store customers according to their purchasing behavior.
Dataset: https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

Task 3:
Develop a Support Vector Machine (SVM) model to classify images of cats and dogs using the Kaggle dataset.
Dataset: https://www.kaggle.com/c/dogs-vs-cats/data
